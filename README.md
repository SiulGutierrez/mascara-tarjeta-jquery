# Máscara para número de tarjeta de crédito/débito

Aplicar máscara a una tarjeta de crédito/débito dejando visibles los últimos 4 dígitos del campo.

Ejemplo:
> Sin máscara:  5110783773817181
> Máscara aplicada:  \*\*\*\*\*\*\*\*\*\*\*\*\*\*\*7181
